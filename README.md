## ğŸ¯ Project Description

This project presents an **interactive retail sales performance dashboard** built using **Microsoft Excel and Power Pivot**.  
The dashboard is designed to support **business-oriented analysis**, focusing on revenue performance, product contribution, store efficiency, and seasonal trends.

Rather than emphasizing visual aesthetics alone, this project prioritizes **analytical clarity and decision-support insights**, simulating a real-world retail performance review scenario.

---

## ğŸ§  Analytical Objectives

The dashboard addresses key business questions such as:

- How does **gross vs nett revenue** evolve over time?
- What is the **revenue gap** caused by returns and discounts?
- Which **products and stores** contribute most to total nett revenue?
- Are there identifiable **seasonal sales patterns**?
- How concentrated is revenue contribution across stores (*Pareto principle*)?
